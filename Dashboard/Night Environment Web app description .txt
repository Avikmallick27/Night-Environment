===========================================
Night Environment
Software Description
===========================================

Programming Languages
---------------------
1. HTML5
2. CSS3
3. JavaScript (ES6)

Software Components
-------------------
1. HTML5
   - Creates the dashboard layout.
   - Displays sensor values, graphs, and sleep records.

2. CSS3
   - Designs the user interface.
   - Provides responsive layout, dark theme, cards, and animations.

3. JavaScript (ES6)
   - Controls dashboard functionality.
   - Reads sensor data from Firebase.
   - Updates graphs in real time.
   - Performs sleep analysis.
   - Saves manual sleep records.

4. Firebase Realtime Database
   - Cloud database for storing sensor readings.
   - Stores manual sleep information.
   - Synchronizes ESP32 and dashboard data.
   - Provides real-time data updates.

5. Firebase Web SDK
   - Connects the web dashboard with Firebase.
   - Reads and writes database data.
   - Monitors Firebase connection status.

6. Chart.js
   - Creates temperature graphs.
   - Creates humidity graphs.
   - Displays historical environmental data.

7. Web Browser
   - Runs the dashboard.
   - Displays live monitoring and graphical analysis.
